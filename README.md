# Training Habit Tracker

週間トレーニングプログラムの継続を支援するための、iPhone向け静的Webアプリケーションです。
PWA (Progressive Web App) に対応しており、iPhone の Safari で「ホーム画面に追加」することで、ネイティブアプリのように全画面かつオフラインでも使用できます。

## 主な機能

- **曜日連動スケジュール自動表示**: 曜日に応じて「今日やるメニュー」を起動直後に表示します（月: 筋トレA、水: 筋トレB、金: 筋トレC、土日: HIIT または 速歩き、その他: 休養日）。
- **ワンタップ簡易記録**: ホーム画面から 1〜2 タップで完了、一部実施、スキップなどを記録できます。
- **最低限メニュー代替モード**: 忙しい日はワンタップで「最低限メニュー」に切り替えられ、記録時にも「代替実施」として区別して保存できます。
- **Web Audio API による案内タイマー**: ワークアウト開始ボタンを押すと、各種目・休憩時間を案内するカウントダウンタイマーが作動します。カウントダウン音・終了アラーム音はブラウザ上で合成再生されるため、通信容量を消費せずオフラインでも鳴ります。
- **8週間進行ガイド**: 現在の経過週数を自動判定（または手動指定）し、週ごとの強度調整の目安を確認できます。
- **データバックアップと初期化**: localStorage に保存されるデータを JSON 形式でコピー・復元可能です。

## GitHub / GitHub Pages での公開手順

本アプリは完全に静的なWebアプリであるため、GitHub Pages で瞬時に無料公開できます。

1. **GitHub リポジトリの作成**
   - GitHub にログインし、新しいリポジトリ（例: `training-habit-tracker`）を **Public** で作成します。

2. **フォルダの移動とプッシュ**
   - 本フォルダの内容をローカルの git で管理するか、作成した GitHub リポジトリにアップロードします。
   - 例（コマンドラインでの操作）:
     ```bash
     git init
     git add .
     git commit -m "Initial commit"
     git branch -M main
     git remote add origin https://github.com/あなたのユーザー名/リポジトリ名.git
     git push -u origin main
     ```

3. **GitHub Pages の有効化**
   - GitHub のリポジトリページを開き、**Settings (設定)** タブを選択します。
   - 左サイドバーの **Pages** メニューを開きます。
   - **Build and deployment** セクションの *Source* で `Deploy from a branch` を選択します。
   - *Branch* を `main` / `/ (root)` に設定し、**Save** をクリックします。
   - 数分待つと、上部に `https://あなたのユーザー名.github.io/リポジトリ名/` のような公開用URLが表示されます。

## iPhone Safari での利用方法

1. **Safariでアクセス**
   - 公開された GitHub Pages の URL に iPhone の **Safari** からアクセスします。

2. **ホーム画面に追加**
   - 画面下部の中央にある **共有ボタン（上矢印のアイコン）** をタップします。
   - メニューをスクロールし、**「ホーム画面に追加」** をタップします。
   - 名前を確認して右上の **「追加」** をタップします。

3. **アプリとして起動**
   - ホーム画面に作成されたアイコンから起動します。
   - Safari のアドレスバーなどが非表示になり、フルスクリーンのWebアプリとして利用できます。
